Das Framework verfügt über eine ganze Reihe integrierter Tests, mit denen die externe Software-Umgebung geprüft werden kann. Alle Tests müssen erfolgreich bestanden werden, damit die Website reibungslos funktionieren kann.

Im Bereich Einstellungen > Tools > Seitenprüfung:

  1. Auf der Registerkarte "Konfigurationsprüfung" müssen alle Anforderungen erfüllt werden. Kommt zu eventuellen Beanstandungen, wenden Sie sich an den Technischen Support von Bitrix.
  2. Auf der Registerkarte "Zugriffsüberprüfung" muss die Option "Komplettprüfung" aktiviert werden, dabei sollen keine Beanstandungen auftreten. Schreibgeschützte Dateien/Ordner müssen außerhalb der Website liegen.

Sollten während des gesamten Tests Beanstandungen aufgetreten sein, müssen diese unbedingt behoben sein, bevor die Website gelauncht wird.
