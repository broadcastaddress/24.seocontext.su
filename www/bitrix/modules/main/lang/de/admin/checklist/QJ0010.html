Bevor die Website gelauchnt wird, muss der Lizenzschlüssel Ihrer Bitrix Edition aktiviert werden. Es wird außerdem empfohlen, die Verfügbarkeit von Updates regelmäßig zu überprüfen, damit man diese rechtzeitig herunterladen und installieren kann. So wird eine stabile und sichere Arbeitsweise der Website sichergestellt und neue Funktionalitäten werden verfügbar.

Werden die Integrationsregeln eingehalten und die Anforderungen dieser Checkliste der Qualitätssicherung erfüllt, wird die Installation der Updates für das Bitrix Framework die Arbeitsweise der Website in keiner Weise beeinträchtigen.

  1. Im Bereich "Einstellungen > Systemeinstellungen > Moduleinstellungen > Hauptmodul" auf der Registerkarte "Update-System" wird überprüft, ob der Lizenzschlüssel eingegeben wurde.

    Auf derselben Registerkarte sollte außerdem die Regelmäßigkeit der Update-Überprüfung (Automatische Überprüfung von Updates) eingestellt. Das ist sehr wichtig, weil mit den Updates diverse Fehlerkorrekturen, Produktverbesserungen sowie neue Funktionalitäten geliefert werden. Die Überprüfung von Updates sollte täglich erfolgen.

  2. Im Bereich "Einstellungen > Aktualisierungen", Unterbereich "Antwort des Update-Servers" werden detaillierte Informationen zum Lizenzschlüssel, zur Produktedition und zum Zeitraum der verfügbaren Updates angezeigt.
  3. Näheres dazu auf der Seite Registrierung und Prüfung des Lizenzschlüssels.
